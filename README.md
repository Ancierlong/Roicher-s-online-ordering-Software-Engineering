Roicher's online ordering

Functionalities
- User Authentication & Management: Secure registration and login system for customers to manage their profiles and track orders.
- Product Content Management (Admin Dashboard): A robust administrative portal featuring full CRUD (Create, Read, Update, Delete) capabilities to manage product listings, images, and descriptions.
- Shopping Cart & Checkout System: An intuitive cart system allowing users to add, modify, and review items before proceeding to checkout.
- GCash Payment Integration (Manual Verification): A local payment workflow where customers can select GCash and upload/paste their transaction receipts for verification.
- Manual Order Tracking: An admin-controlled order status system to update milestones (e.g., Pending, Paid, Shipped, Delivered) and keep customers informed.
